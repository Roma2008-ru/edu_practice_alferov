# 5. Построение сети ip телефонии между удаленными маршрутизаторами

Строим топологию сети

Настраиваем Serial интерфейс на маршрутизаторах

Настраиваем маршрутизацию по протоколу EIGRP на маршрутизаторе

Выполняем проверку связи между маршрутизаторами

Создаем подинтерфейсы для VLAN

Изменяем имя коммутатора

Отключаем синтаксис ввода слов от DNS серверов

Настраиваем пароли для защиты коммутатора

Настраиваем первый порт коммутатора в транковый режим

Настраиваем порты коммутатора на access vlan
